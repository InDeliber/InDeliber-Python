# 3.2 数学函数


1. Python中有一个数学模块，提供了大部分常用的数学函数。

2. 模块（module）是指一个**包含相关函数集合的文件**

3. 模块导入语句：import  module_name
   + `import math`

4. 通过模块的**导入语句（import statement）**导入模块后，会生成一个名为“模块名”的**模块对象（module object）**，可以使用打印语句，打印这个对象获得关于它的一些信息。
   `import math`
   `print(math)`
   + `math <module 'math' (built − in)>`

5. 模块对象包含了，**定义在模块内的所有函数和变量**想要访问其中的函数或者变量需要，**指定模块的名字、函数/变量名、使用点号分隔**，这种形式被称为**点标记法（dot notation）**
   + `module_name.functiont_name()`
